# VIT Carpool

## Overview

VIT Carpool is a modern web application designed to facilitate carpooling within the VIT (Vellore Institute of Technology) community. It aims to connect students and faculty who are traveling to the same destination, helping them to share rides, reduce travel costs, and contribute to a more sustainable environment.

The application is built with a focus on a smooth user experience, leveraging modern web technologies and a robust backend.

## Features

- **User Authentication**: Secure user login and registration.
- **Ride Creation**: Users can create new ride offerings, specifying details like origin, destination, time, date, and available seats.
- **Ride Search & Joining**: Users can search for available rides based on their travel needs and join them.
- **Ride Management**: Functionality for users to view, edit, and cancel their created rides.
- **Joined Rides Overview**: Users can see a list of rides they have joined.
- **Real-time Updates**: Integration with WhatsApp for communication between ride participants.
- **Responsive Design**: A user interface that adapts to various screen sizes.
- **Robust State Management**: Utilizes React Context for managing authentication and ride-related data.

## Technologies Used

- **Frontend**:
  - **React**: A JavaScript library for building user interfaces.
  - **TypeScript**: A typed superset of JavaScript that enhances code quality.
  - **Vite**: A fast build tool for a lightning-fast development experience.
  - **Tailwind CSS**: A utility-first CSS framework for rapid UI development.
  - **Shadcn UI**: Used for consistent design.
- **Backend**:
  - **Supabase**: Open-source Firebase alternative providing PostgreSQL, Auth, and APIs.
